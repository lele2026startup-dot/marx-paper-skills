# GB/T 7714-2015 著录格式参考

编写脚注和参考文献时查阅本文件。脚注带页码，参考文献不带页码，其余字段一致。

## 文献类型标识

| 类型 | 标识 | 类型 | 标识 |
|------|------|------|------|
| 专著 | M | 期刊 | J |
| 论文集 | C | 学位论文 | D |
| 报纸 | N | 报告 | R |
| 标准 | S | 专利 | P |
| 析出文献(单篇) | A | 其他 | Z |
| 电子文献 | EB/OL | 数据库 | DB |

## 一、专著（含编著、译著）

**格式**：`[序号] 主要责任者.文献题名[M].出版地：出版者，出版年.起止页码.`

**示例**：
```
[1] 唐绪军. 报业经济与报业经营[M]. 北京: 新华出版社, 1999: 117-121.
[2] 陆学艺主编. 当代中国社会流动[M]. 北京：社会科学文献出版社, 2004: 48-49.
[3] 霍布斯. 利维坦[M]. 黎思复,黎廷弼译. 北京：商务印书馆, 1985: 132.
```

**注意**：
- 编著/译著在编者、译者姓名后加"编""主编""编著"或"译"
- 著者超三人，只写前三人，后加"等"或"et al"
- 出版地后用全角冒号"："
- 脚注标页码，参考文献不标

## 二、期刊文章

**格式**：`[序号] 主要责任者.文献题名[J].刊名，年，卷(期)：起止页码.`

**示例**：
```
[1] 何龄修. 读顾城《南明史》[J]. 中国史研究, 1998, (3): 167-173.
[2] 张小明. 革命、修正和补充[J]. 欧洲, 2001(3): 16.
```

**注意**：区分卷和期。有卷号写"年,卷(期)"，无卷号写"年,(期)"。

## 三、学位论文

**格式**：`[序号] 作者.题名[D].授学位地:授学位单位,授学位年:起止页码.`

**示例**：
```
[1] 苏长和. 全球公共问题与国际合作[D]. 上海：复旦大学, 1999: 55.
```

## 四、会议论文

**格式**：`[序号] 作者.篇名[C].会议名,会址,开会年:起止页.`

## 五、报纸文章

**格式**：`[序号] 主要责任者.文献题名[N].报纸名，出版日期(版次).`

主标题与副标题之间用冒号，不用破折号。

## 六、文集中的析出文献

**格式**：`[序号] 析出文献主要责任者.析出文献题名[A].原文献主要责任者.原文献题名.出版地：出版者，出版年.起止页码.`

## 七、重复引用

**连续引用同一文献**：
- 中文：`同上（书），第╳页.`（注意这里保留"第"和"页"）
- 外文：`Ibid., p. ×.`

**非连续引用**（中间隔了别的引用）：
- 只注著者、文献名、页码，不用完整格式
- 中文例：`亚里士多德. 政治学, 56.`
- 外文例：`Alexander Wendt, Social Theory of International Politics, p.18.`

## 八、转引

先按正常方式著录被转引的文献，以句号结束；再以"转引自"引出来源文献。

```
张维华. 明史欧洲四国传注释[M]. 上海：上海古籍出版社, 1982: 173. 转引自陈乐民. 陈乐民集[M]. 北京：中国社会科学出版社, 2002: 205.
```

## 九、互联网资料

**格式**：`[序号] 主要责任者.电子文献题名[EB/OL].出处或地址, 发表或更新日期/引用日期.`

**注意**：
- 未经第三方审核的网络百科（维基、百度百科）不能作为学术引用
- 必须注明网址和登录时间

## 十、参考文献表编排规则

1. 顺序编码制：按文献在论文中**首次引用的顺序**排列
2. 序号左顶格，[1][2][3]...
3. 每条以"."结束
4. 作者姓名写到第三位，余者写"等"或"et al"
5. 汉语拼音姓名先姓后名
6. **不标页码**
7. **不重复**（同一文献多处引用只列一次）
8. **未引用的不得出现**
