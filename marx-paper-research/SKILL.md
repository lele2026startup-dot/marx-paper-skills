# 马克思主义理论学科论文素材整理

## 这个 skill 解决什么问题

选题定了、文献找来了（通常 25-30 篇，由 marx-paper-topic skill 提醒用户去找），接下来要把这些文献里能用的东西提取出来。马理论科论文里最核心的素材是**原话引用**——领导人怎么说、马克思恩格斯怎么说、列宁怎么说、毛泽东怎么说、国外学者怎么说。把这些原话连同它们的**出处标注信息**（哪本书、哪个版本、哪一页）一起提取出来，整理成分类素材库，后面写论文时就能直接复用，不用再翻原书。

为什么复用别人用过的原话是合理做法：别人论文里引用的原话，已经带好了脚注或参考文献的完整出处信息（页码、版本等）。把这些原话+出处一起提取出来，再用到自己论文里，出处信息可以直接复用——大家都这么做，等于站在前人肩膀上，不必每条引用都从头去原书核实页码。复用的出处信息最后由 marx-paper-format skill 编排成规范脚注和参考文献，并在完成后告知用户哪些是复用的。

## 工作流程

### 第一步：读文献

用户找来的文献可能是 PDF 或 MD：
- **MD**：直接读，最省事。建议用户尽量转成 MD，但不强求。
- **PDF**：用 pdf skill 读（pdfplumber 提取文本）。PDF 也能读，只是慢一些。

### 第二步：跟用户确认分类

马理论科论文里常见几类原话来源，按来源分文件夹整理。常见这几类：

- **领导人原话**（党和国家领导人的讲话、文章、著作里的原话）
- **马克思恩格斯原话**
- **列宁原话**
- **毛泽东原话**
- **国外学者原话**（非马克思主义的国外学者，如西方马克思主义各流派、其他国外人文社科学者）

但具体情况具体判断——不是每篇文献都会出现所有这些来源，也可能有其他来源（比如其他经典作家、其他中国学者）。所以整理前**先跟用户商量**：根据这批文献的实际情况，要分几类、每类叫什么，让用户确认。别擅自定死分类，也别漏掉用户实际需要的类别。

### 第三步：建文件夹并整理

用户确认分类后，在工作区建一个素材整理文件夹，下面按分类建子文件夹。**建之前征得用户同意**，别擅自创建目录。

每篇文献读完后，把里面的原话引用提取到对应分类。每条原话要记录两样东西：

1. **原话内容**本身
2. **出处标注信息**：谁说的/写的、出自哪本书或哪篇文章、哪个版本（出版社、出版年、译者等）、哪一页——也就是别人论文里给这条原话配的脚注或参考文献信息

为什么出处信息也要提取：后面 marx-paper-format skill 写脚注和参考文献时需要这些信息，尤其页码和版本。别人已经标好出处了，直接拿来用，不用再翻原书逐条核实。

### 第四步：产出素材库

整理完，每个分类文件夹里是该类原话的集合，每条都带出处信息。这个素材库就是写作时的"弹药库"——写论文时哪句原话合适就用哪句，出处信息现成可用。

## 衔接

整理完素材库后，衔接 **marx-paper-format**（格式）skill：写作时引用这些原话，format skill 会把出处信息编排成规范的脚注（圈码 ①②③、每页重编号、GB/T 7714 格式）和参考文献表。其中**复用整理素材里原话的引用**，format skill 不必再去原 PDF 核实页码（别人已核实过），但会在完成后于对话里告知用户哪些是复用的，让用户心里有数。
