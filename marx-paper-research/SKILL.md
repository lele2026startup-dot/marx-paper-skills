# 马克思主义理论学科论文素材整理

## 这个 skill 解决什么问题

选题定了、文献找来了（通常 25-30 篇，由 marx-paper-topic skill 提醒用户去找），结构也定好了（由 marx-paper-structure skill 搭好——整理间接引用时要对照结构判断有没有用，所以结构得先定），接下来要把这些文献里能用的东西提取出来。

文献里的引用分两种，整理方式不同：

- **直接引用**：带双引号的、标了具体出处的原文。不只是语录式的讲话原话，也包括著作、文章里的话。按来源（领导人/马恩/列宁/毛泽东/国外学者等）分文件夹，全部整理。
- **间接引用**：没有双引号但标了脚注/尾注的转述。不全整理——对照结构判断每条有没有用，有用的才整理，按结构的各部分分文件夹，每篇参考论文最多取 1 条。

两种引用都要连同**出处标注信息**（哪本书、哪个版本、哪一页）一起提取，后面写论文时复用，不用再翻原书。

为什么复用别人用过的原话是合理做法：别人论文里引用的原话，已经带好了脚注或参考文献的完整出处信息（页码、版本等）。把这些原话+出处一起提取出来，再用到自己论文里，出处信息可以直接复用——大家都这么做，等于站在前人肩膀上，不必每条引用都从头去原书核实页码。复用的出处信息最后由 marx-paper-format skill 编排成规范脚注和参考文献，并在完成后告知用户哪些是复用的。

## 工作流程

### 第一步：读文献

用户找来的文献可能是 PDF 或 MD：
- **MD**：直接读，最省事。建议用户尽量转成 MD，但不强求。
- **PDF**：用 pdf skill 读（pdfplumber 提取文本）。PDF 也能读，只是慢一些。

### 第二步：跟用户确认直接引用的分类

直接引用按来源分文件夹整理。注意"原话"不只是讲话的语录式原话，也包括著作、文章里的话——只要是带双引号、标了具体出处的原文，都算直接引用。常见这几类来源：

- **领导人原话**（党和国家领导人的讲话、文章、著作里的原话）
- **马克思恩格斯原话**
- **列宁原话**
- **毛泽东原话**
- **国外学者原话**（非马克思主义的国外学者，如西方马克思主义各流派、其他国外人文社科学者）

但具体情况具体判断——不是每篇文献都会出现所有这些来源，也可能有其他来源（比如其他经典作家、其他中国学者）。所以整理前**先跟用户商量**：根据这批文献的实际情况，要分几类、每类叫什么，让用户确认。别擅自定死分类，也别漏掉用户实际需要的类别。

### 第三步：整理直接引用

用户确认分类后，在工作区建一个直接引用素材文件夹，下面按分类建子文件夹。**建之前征得用户同意**，别擅自创建目录。

每篇文献读完后，把里面的直接引用提取到对应分类。每条要记录两样东西：

1. **原话内容**本身
2. **出处标注信息**：谁说的/写的、出自哪本书或哪篇文章、哪个版本（出版社、出版年、译者等）、哪一页——也就是别人论文里给这条原话配的脚注或参考文献信息

为什么出处信息也要提取：后面 marx-paper-format skill 写脚注和参考文献时需要这些信息，尤其页码和版本。别人已经标好出处了，直接拿来用，不用再翻原书逐条核实。

### 第四步：整理间接引用

间接引用是参考论文里没有双引号但标了脚注/尾注的转述——读到小标（脚注/尾注标记）就知道前面那句是引用，再看有没有双引号判断是直接还是间接，没有双引号就是间接。

间接引用**不全整理**，要对照 marx-paper-structure skill 定好的结构判断每条有没有用：

1. 读到间接引用，先判断它对应结构的哪个部分、对自己论点有没有帮助
2. 有用的才整理：把参考论文里那段间接引用的原段截取下来，连同它标的出处信息（脚注/参考文献）一起记录
3. 没用的跳过，不用全捞

整理出来的间接引用，按**结构的各个部分**分子文件夹存放（比如结构有三大标题，就分三个子文件夹；每个子文件夹里放对应该部分能用的间接引用）。这样写作时哪个部分需要间接引用支撑，直接去对应文件夹取。

**每篇参考论文最多取 1 条间接引用**——避免过度依赖单篇文献，保证素材来源分散。25-30 篇文献各取最多 1 条，间接引用素材量也够用了。

写作时复用间接引用的注意事项（由 marx-paper-write 执行）：必须换说法，不能照抄参考论文原段；不强行复用，不贴合论点就不用。

### 第五步：产出素材库

整理完，直接引用各分类文件夹 + 间接引用各结构部分文件夹，组成完整的素材库，每条都带出处信息。这个素材库就是写作时的"弹药库"——直接引用哪句原话合适就用哪句，间接引用哪段能支撑论点就换说法用，出处信息现成可用。

## 衔接

整理完素材库后，衔接 **marx-paper-write**（写作）skill 撰写正文：直接引用直接用、出处信息复用，间接引用换说法用、每篇最多 1 条。写作完成后再衔接 **marx-paper-format**（格式）skill：format skill 会把出处信息编排成规范的脚注（圈码 ①②③、每页重编号、GB/T 7714 格式）和参考文献表。其中**复用整理素材里原话的引用**，format skill 不必再去原 PDF 核实页码（别人已核实过），但会在完成后于对话里告知用户哪些是复用的，让用户心里有数。
